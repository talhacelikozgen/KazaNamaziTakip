# KazaNamaziTakip
Kaza namazı borcunuz mu var, takip etmekte güçlük mü çekiyorsunuz, hiç endişelenmeyin KazaNamazıTakip sizlerle. İhtiyacınıza göre hesaplama yapın, her kaza kıdığınızda listeyi doldurun, o sizin için takip etsin. Allahü Teala hepimize kolaylıklar ve güzellikler ihsan etsin, amin. (Kitaplar İçin: https://www.hakikatkitabevi.net)
